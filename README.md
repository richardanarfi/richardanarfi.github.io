# Richard Anarfi GitHub Pages Starter

## What you edit most
Open `src/content.js` and update:
- name, title, subtitle
- GitHub, LinkedIn, Scholar links
- projects
- writing topics
- about text
- contact email

Replace `public/resume.pdf` with your real resume PDF.

## Run locally
```bash
npm install
npm run dev
```

## Deploy on GitHub Pages
1. Create a repo named `richard-anarfi.github.io`
2. Upload these files to the repo
3. Push to `main`
4. In GitHub repo Settings > Pages, choose `GitHub Actions`
5. Wait for the workflow to finish

Your site will appear at `https://richard-anarfi.github.io`
