# Static portfolio starter

This is a plain HTML/CSS GitHub Pages portfolio.

## Files
- `index.html` — homepage
- `assets/styles.css` — shared styles
- `assets/site.js` — small mobile nav script
- `projects/` — project pages
- `writing/` — essay pages
- `images/` — screenshots and graphics
- `resume.pdf` — replace with your real resume

## GitHub Pages settings
Use:
- **Deploy from a branch**
- Branch: `main`
- Folder: `/ (root)`

## Easiest edits
1. Replace the placeholder links and email in `index.html`
2. Upload your real `resume.pdf`
3. Replace the text inside the project pages
4. Add screenshots into `images/`
