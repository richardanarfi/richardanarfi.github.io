Flagship benchmark project

AgentEval

A benchmark harness for tool-using LLM systems, focused on task success, tool selection accuracy, failure recovery, and robustness across prompts and models.

  • Benchmark tasks for retrieval, structured extraction, tool use, and recovery behavior
  • Comparable runs across models and prompt variants
  • Failure analysis and metrics reporting

Synthetic data and validation

SynthBench

A pipeline for generating synthetic agent tasks, alternate-valid answers, and evaluation rubrics, with checks for whether synthetic data is actually useful.

  • Human seed set plus synthetic expansion
  • Rubric generation and alternate-answer support
  • Validation against human-authored tasks
